# Backrooms Builder

The Backrooms builder is an editor-friendly level assembly tool for quickly blocking out yellow-room levels with gameplay markers.

## Files

- Builder script: `res://scripts/backrooms_builder.gd`
- Demo scene: `res://scenes/backrooms/backrooms_builder_demo.tscn`
- Kit scenes: `res://scenes/backrooms/kit/`
- Reusable exit: `res://scenes/common/level_exit_basic.tscn`
- Reusable watcher: `res://scenes/common/watcher_monster_basic.tscn`
- Reusable chaser: `res://scenes/common/chaser_monster_basic.tscn`
- Reusable pressure plate: `res://scenes/common/pressure_plate_basic.tscn`

## Layout Symbols

| Symbol | Result |
| --- | --- |
| `#` | Wall block |
| `.` | Floor and ceiling |
| `L` | Floor, ceiling, and flickering fluorescent light |
| `S` | Floor, ceiling, and spawn marker |
| `E` | Floor, ceiling, exit marker, and `LevelExit` |
| `N` | Floor, ceiling, and generated note |
| `W` | Floor, ceiling, and watcher monster |
| `C` | Floor, ceiling, and sprint-hearing chaser monster |
| `B` | Floor, ceiling, and low barrier/cover |
| `P` | Floor, ceiling, and pressure plate |

Generated content is grouped under `GeneratedBackrooms` as `Geometry`, `Markers`, `Mechanics`, `Notes`, and `Monsters`.

Pressure plates are latch-once by default, so they are compatible with solo local testing. Disable `latch_once` on a placed plate when a level needs a true hold-the-switch co-op gate.

The reusable chaser generated by `C` starts dormant, activates after the first collected note, patrols a small area when it has no target, and triggers a short threat cue. Set `notes_required_to_activate` to `0` on a placed chaser if it should start on a timer immediately.

## Workflow

1. Create or duplicate a Backrooms level scene.
2. Add a `Node3D` with `BackroomsBuilder`.
3. Edit the multiline `layout` string in the inspector.
4. Keep `rebuild_in_editor` enabled while blocking out the map.
5. Use `N`, `W`, and `C` sparingly; the level should remain readable and not overloaded.
6. Run a local scene smoke check before using the level in `Main`.
